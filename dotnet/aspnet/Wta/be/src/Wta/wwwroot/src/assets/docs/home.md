# 虚拟工厂抽象与建模

## 组织机构

### 基本职能

```mermaid
flowchart TD

组织-->财务
组织-->生产运作
组织-->营销
```

### 生产与运作系统

```mermaid
flowchart LR

供应商-->投入-->转换过程-->产出-->客户(客户/市场)
转换过程-->控制
控制-->转换过程
控制-->投入
控制-->产出
```

```mermaid
flowchart TD

组织-->总经办

营销中心-->业务部
营销中心-->市场部

总经办-->技术中心

技术中心-->工艺部
总经办-->生产中心

生产中心-->生产车间
生产中心-->采购部
生产中心-->仓储物流部

总经办-->检测中心
检测中心-->品管部

总经办-->行政中心
行政中心-->人事部
行政中心-->办公室
行政中心-->网络技术部

总经办-->财务中心
财务中心-->财务部
```
